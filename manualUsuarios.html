Manual para Crear un Usuario y Proteger su Espacio de Trabajo

Introducción

Este manual proporciona instrucciones detalladas para crear un usuario en sistemas operativos Windows y Linux, asegurando que su espacio de trabajo (su carpeta personal) esté protegido del acceso de otros usuarios no autorizados.

Creación de Usuario en Windows (Local)

Pasos para crear un usuario (Método Gráfico)

  1. Accede a Configuración (presionando Win + I).
  2. Haz clic en Cuentas.
  3. Selecciona Familia y otros usuarios (u Otros usuarios).
  4. Bajo "Otros usuarios", haz clic en Agregar a otra persona a este PC.
  5. Selecciona No tengo los datos de inicio de sesión de esta persona.
  6. Luego selecciona Agregar un usuario sin cuenta de Microsoft.
  7. Introduce un nombre de usuario (ej. invitado), una contraseña segura y completa las preguntas de seguridad.
  8. Haz clic en Siguiente. El usuario será creado.

Pasos para crear un usuario (Línea de Comandos)

  1. Abre PowerShell o CMD como Administrador.
  2. Ejecuta el siguiente comando para crear el usuario (cambia nombre_usuario y ContraseñaSegura!):
    net user nombre_usuario ContraseñaSegura! /add
  3. Esto crea el usuario y su carpeta de perfil (C:\Users\nombre_usuario) la primera vez que inicie sesión.

Proteger el Espacio de Trabajo

  • Tipo de Cuenta (Usuario Estándar): Los usuarios creados así son Usuarios Estándar, lo que les impide instalar software o acceder a los archivos de otros.
  • Permisos NTFS: Windows protege automáticamente la carpeta C:\Users\nombre_usuario, dando control total solo al usuario y a los administradores.
  • Cifrado (BitLocker): En Windows Pro o Enterprise, activa BitLocker para cifrar todo el disco duro.

Creación de Usuario en Linux (Ubuntu/Debian)

Pasos para crear un usuario

  1. Abre una Terminal (Ctrl + Alt + T).
  2. Ejecuta:
    sudo adduser nombre_usuario
  3. Introduce tu contraseña de administrador (sudo).
  4. Define la contraseña para el nuevo usuario y confirma.
  5. Completa la información adicional o presiona Enter para omitirla.
  6. Confirma escribiendo "S".

Proteger el Espacio de Trabajo

  • Permisos del Directorio Home: Por defecto, /home/nombre_usuario suele tener permisos 755. Para hacerlo privado:
    sudo chmod 700 /home/nombre_usuario
    (700 = solo el propietario tiene acceso).
  • Configuración de Umask: Para que los nuevos archivos sean privados, edita /home/nombre_usuario/.bashrc y agrega:
    umask 077
  • Firewall (UFW): Activa el firewall básico de Ubuntu:
    sudo ufw enable
  • Cifrado (LUKS / eCryptfs): Durante la instalación, Ubuntu permite cifrar el disco completo (LUKS) o solo la carpeta personal (eCryptfs).
Nota: La protección del espacio de trabajo se basa en los permisos del sistema de archivos (NTFS en Windows y POSIX en Linux), que impiden el acceso no autorizado a archivos privados.

Conclusión

Siguiendo estos pasos, podrás crear usuarios en Windows y Linux de forma segura. La clave está en usar cuentas estándar y asegurar que las carpetas personales sean privadas mediante permisos correctos y, opcionalmente, cifrado.
